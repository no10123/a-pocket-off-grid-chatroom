## How It Works

## How To Use It

you cant really use it unless you build it yourelf as you cant connect to the esp32's ip, on the online verison, but it should work in theory, its a simple chatroom, you have 2 input feilds, one for username the other for msgs, click enter to send, also  is spam proof, and you can change color of your msgs, with the color piker.
